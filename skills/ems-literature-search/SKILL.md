# Поиск литературы по ЭМС

Используй этот навык, когда нужно собрать научную литературу по влиянию диссоциации и рекомбинации молекул воды на перенос в электромембранных системах.

## Порядок работы

1. Уточни поисковую формулу на русском и английском языках.
2. Проверь основные базы: Scopus, Web of Science, arXiv.org, eLIBRARY.
3. Подключи навык [firecrawl-scraping-pipeline](../firecrawl-scraping-pipeline/SKILL.md) для открытого поиска, извлечения страниц, проверки DOI и расширения подборки.
4. Через Firecrawl MCP/CLI выполни открытый поиск и извлечение страниц:
   - `firecrawl-search` для первичного списка страниц;
   - `firecrawl-scrape` для страниц статей, препринтов и журналов;
   - `firecrawl-map` для поиска нужных URL внутри сайта журнала, выпуска или лаборатории;
   - `firecrawl-crawl` для ограниченного обхода небольшого раздела сайта;
   - `firecrawl-interact` для динамических страниц, пагинации и форм;
   - `firecrawl-parse` для законно полученных локальных PDF, HTML и выгрузок;
   - `firecrawl-agent` только для узкого структурированного извлечения.
5. Для каждой найденной работы собери: название, авторов, год, журнал, DOI, ссылку, тип исследования, доступность полного текста.
6. Переведи название и аннотацию на русский язык.
7. Сделай краткое резюме и научный анализ.
8. Сгруппируй работы по темам: диссоциация воды, рекомбинация, перенос ионов, электроконвекция, предельные и сверхпредельные токи, мембранные интерфейсы.
9. Сформируй итоговую таблицу и краткий вывод.

## Критерии качества

- DOI не должен быть выдуман.
- Нельзя выдавать платную или закрытую статью за скачанную.
- Нельзя смешивать электролиз, обычную мембранную фильтрацию и электромембранные системы без пояснения.
- Нельзя считать результат веб-скрейпинга полноценной заменой записи в Scopus или Web of Science.
- Важнее точность и проверяемость, чем большое число слабых ссылок.

# Поиск литературы по ЭМС

Используй этот навык, когда нужно собрать научную литературу по влиянию диссоциации и рекомбинации молекул воды на перенос в электромембранных системах.

## Порядок работы

1. Уточни поисковую формулу на русском и английском языках.
2. Проверь основные базы: Scopus, Web of Science, arXiv.org, eLIBRARY.
3. Через Firecrawl MCP выполни открытый поиск и извлечение страниц.
4. Для каждой найденной работы собери: название, авторов, год, журнал, DOI, ссылку, тип исследования, доступность полного текста.
5. Переведи название и аннотацию на русский язык.
6. Сделай краткое резюме и научный анализ.
7. Сгруппируй работы по темам: диссоциация воды, рекомбинация, перенос ионов, электроконвекция, предельные и сверхпредельные токи, мембранные интерфейсы.
8. Сформируй итоговую таблицу и краткий вывод.

## Критерии качества

- DOI не должен быть выдуман.
- Нельзя выдавать платную или закрытую статью за скачанную.
- Нельзя смешивать электролиз, обычную мембранную фильтрацию и электромембранные системы без пояснения.
- Важнее точность и проверяемость, чем большое число слабых ссылок.
